## 🚀 Cara Menjalankan Secara Lokal

```bash
# 1. Clone repo ini
git clone https://github.com/Wahid1178/portfolio.git
cd portfolio

# 2. Install dependencies
npm install

# 3. Jalankan development server
npm run dev

# 4. Buka di browser
# http://localhost:5173
```

---

## ☁️ Deploy ke Vercel

```bash
# Push ke GitHub terlebih dahulu
git add .
git commit -m "feat: update portfolio"
git push origin main
```

Lalu buka [vercel.com](https://vercel.com) → **Add New Project** → Import repo → Framework **Vite** → **Deploy** ✅
